Screens

Pictor applies simple semantics when dealing with grids and creating utilities. Screens can be broken down into roughly 5 devices:

  • mobile: 600px default
  • tablet: 900px default
  • laptop: 1200px default
  • desktop: 1400px default
  • hd: 1700px default
